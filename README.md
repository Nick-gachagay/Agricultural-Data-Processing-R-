 Agricultural Data Processing Project (R)

 Overview
This project demonstrates data cleaning, standardization, and analysis of agricultural datasets using R.

 Objectives
- Load and explore agricultural data
- Clean and standardize datasets
- Perform basic analysis
- Generate outputs for reporting

 Tools Used
- R
- dplyr
- ggplot2
- readr

 Project Structure
- data/: raw and processed datasets
- scripts/: R scripts for each workflow stage
- outputs/: generated plots and tables

 How to Run
1. Run scripts/01_data_loading.R
2. Run scripts/02_data_cleaning.R
3. Run scripts/03_analysis.R

 Sample Output
- Clean dataset
- Average yield table
- Yield trend visualization
